# 🚀 Быстрый старт Biolab

## После установки MongoDB

### Шаг 1: Запуск проекта
```powershell
cd C:\bat\biolab-main
.\start.ps1
```

Этот скрипт:
- Запустит MongoDB (если не запущен)
- Запустит Node.js сервер на порту 3000

### Шаг 2: Создание администратора
Откройте новое окно PowerShell:
```powershell
cd C:\bat\biolab-main
.\create-admin.ps1
```

Введите логин и пароль (или оставьте по умолчанию: admin/admin123)

### Шаг 3: Вход в админку
Откройте браузер:
```
http://localhost:3000/admin/login.html
```

Войдите с созданными учётными данными.

### Шаг 4: Добавьте категории
В админке перейдите в "Категории" и создайте:
- Деревья (trees)
- Кустарники (shrubs)
- Цветы (flowers)

### Шаг 5: Добавьте товары
В разделе "Товары":
1. Нажмите "Добавить товар"
2. Заполните форму
3. Загрузите изображение
4. Сохраните

### Шаг 6: Проверьте фронтенд
```
http://localhost:3000/
```

## Полезные ссылки

- **Фронтенд:** http://localhost:3000/
- **Админка:** http://localhost:3000/admin/login.html
- **API документация:** см. README.md

## Структура админки

### Главная
- Статистика по товарам и заказам
- Последние заказы
- Товары с низким остатком

### Товары
- Просмотр всех товаров
- Добавление/редактирование/удаление
- Загрузка изображений
- Фильтры и поиск

### Заказы
- Просмотр всех заказов
- Детали заказа с QR-кодом
- Выдача заказа (смена статуса)
- Отмена заказа
- Сканирование QR

### Категории
- Просмотр категорий
- Добавление/удаление

## Тестирование заказа

1. Откройте фронтенд (когда будет готов)
2. Выберите товар
3. Нажмите "Заказать"
4. Заполните форму
5. Получите QR-код на email
6. В админке найдите заказ
7. Отметьте как "Выдан"

## Остановка сервера

Нажмите `Ctrl+C` в окне PowerShell где запущен сервер.

## Проблемы?

**Порт 3000 занят:**
```powershell
# Измените PORT в .env на другой (например, 3001)
```

**MongoDB не запускается:**
```powershell
# Проверьте, запущен ли процесс
Get-Process mongod

# Запустите вручную
mongod
```

**Забыли пароль админа:**
```powershell
# Подключитесь к MongoDB и удалите коллекцию admins
# Затем создайте нового админа через create-admin.ps1
```
