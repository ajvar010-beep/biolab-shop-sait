## 🌐 Вариант 2: Доступ из интернета (через ngrok)

Если заказчик в другом месте, используй ngrok для публичного доступа.

### Установка ngrok:

1. Скачай: https://ngrok.com/download
2. Распакуй в любую папку
3. Зарегистрируйся на ngrok.com (бесплатно)
4. Получи authtoken в личном кабинете
5. Выполни: `ngrok config add-authtoken ТВО_ТОКЕН`

### Запуск ngrok:

```bash
ngrok http 3000
```

Ты получишь публичный URL вида: `https://abc123.ngrok-free.app`

Этот URL можно отправить заказчику - он будет работать из любой точки мира!

**Важно:**
- Ноутбук должен быть включен
- Интернет должен работать
- Бесплатный ngrok работает 2 часа, потом нужно перезапустить

## 🔥 Вариант 3: Localtunnel (альтернатива ngrok)

Если не хочешь регистрироваться:

```bash
# Установка
npm install -g localtunnel

# Запуск
lt --port 3000
```

Получишь URL вида: `https://random-name.loca.lt`

## ⚠️ Важные моменты:

**Файрвол Windows:**
Если не работает доступ из сети, разреши Node.js в файрволе:
1. Панель управления → Брандмауэр Windows
2. Дополнительные параметры → Правила для входящих подключений
3. Создать правило → Для программы → node.exe
4. Разрешить подключение

**Спящий режим:**
Отключи спящий режим на время демо:
- Параметры → Система → Питание и спящий режим
- Поставь "Никогда"

**Антивирус:**
Если антивирус блокирует - временно отключи или добавь исключение.
