# Руководство пользователя Biolab

## Для покупателей

### Как сделать заказ

1. Откройте сайт биолаборатории
2. Просмотрите каталог растений
3. Используйте поиск для быстрого нахождения нужного товара
4. Нажмите на карточку товара для просмотра деталей и фото
5. Нажмите кнопку **"Заказать"** на понравившемся товаре
6. Заполните форму заказа:
   - Ваше имя
   - Телефон для связи
   - Email (на него придет QR-код)
   - Количество товара
7. Нажмите **"Оформить заказ"**
8. Сохраните QR-код (скриншот или из письма на email)
9. Приходите в школьную теплицу с QR-кодом
10. Покажите QR-код администратору
11. Оплатите и получите ваш заказ

### Важно знать

- **Оплата**: только при получении (наличные или перевод)
- **QR-код**: обязательно сохраните его, он нужен для получения заказа
- **Email**: проверьте папку "Спам", если письмо не пришло
- **Остатки**: на сайте показано актуальное количество товара
- **Отмена**: для отмены заказа свяжитесь с администратором

### Настройки сайта

**Тема оформления:**
- Меню → Настройки темы → Тёмная/Светлая

**Размер текста:**
- Меню → Размер текста → Мелкий/Нормальный/Крупный

## Для администраторов

### Вход в админку

1. Откройте `https://ваш-сайт.com/admin/login.html`
2. Введите логин и пароль
3. Нажмите "Войти"

### Управление товарами

**Добавление товара:**
1. Админка → Товары → "Добавить товар"
2. Заполните поля:
   - Название
   - Описание
   - Цена (в рублях)
   - Категория
   - Остаток (количество)
   - Размер карточки (обычная/широкая/большая)
3. Загрузите изображение (необязательно)
4. Добавьте ссылку на 3D модель (необязательно)
5. Нажмите "Сохранить"

**Редактирование товара:**
1. Найдите товар в списке
2. Нажмите "Редактировать"
3. Измените нужные поля
4. Нажмите "Сохранить"

**Удаление товара:**
1. Найдите товар в списке
2. Нажмите "Удалить"
3. Подтвердите удаление

### Управление заказами

**Просмотр заказов:**
- Админка → Заказы
- Фильтр по статусу: Все/Новые/Выполненные/Отменённые

**Выдача заказа:**
1. Покупатель показывает QR-код
2. Отсканируйте QR-код (кнопка "Сканировать QR")
3. Или введите номер заказа вручную
4. Проверьте детали заказа
5. Выдайте товар покупателю
6. Получите оплату
7. Нажмите "Отметить как выданный"

**Отмена заказа:**
1. Найдите заказ в списке
2. Нажмите "Отменить"
3. Товары автоматически вернутся на склад

### Управление категориями

**Добавление категории:**
1. Админка → Категории → "Добавить категорию"
2. Введите название (например, "Цветы")
3. Нажмите "Сохранить"

**Редактирование/Удаление:**
- Аналогично управлению товарами

### Статистика

На главной странице админки отображается:
- Общее количество товаров
- Количество новых заказов
- Количество выполненных заказов
- Общая сумма продаж

### Советы по работе

**Фотографии товаров:**
- Используйте качественные фото (минимум 800x600px)
- Фотографируйте на светлом фоне
- Показывайте товар с разных ракурсов
- Размер файла: до 5MB

**Описания товаров:**
- Укажите размер растения
- Опишите особенности ухода
- Добавьте интересные факты
- Используйте простой язык

**Управление остатками:**
- Обновляйте остатки регулярно
- При получении новых растений добавляйте их в систему
- Товары с нулевым остатком автоматически помечаются как "Нет в наличии"

**Работа с заказами:**
- Проверяйте новые заказы ежедневно
- Связывайтесь с покупателями при необходимости
- Выполняйте заказы в порядке поступления
- Отменяйте заказы, если товар закончился

### QR-сканер

**Использование встроенного сканера:**
1. Админка → Заказы → "Сканировать QR"
2. Разрешите доступ к камере
3. Наведите камеру на QR-код
4. Система автоматически найдет заказ

**Если сканер не работает:**
- Введите номер заказа вручную (формат: BL1234567890123)
- Номер заказа указан в письме покупателя

### Безопасность

**Пароль:**
- Используйте надежный пароль (минимум 8 символов)
- Не сообщайте пароль посторонним
- Меняйте пароль раз в 3 месяца

**Выход из системы:**
- Всегда выходите из админки после работы
- Особенно на общих компьютерах

**Резервное копирование:**
- Данные автоматически сохраняются в облаке
- Для дополнительной безопасности делайте экспорт данных раз в месяц

## Техническая поддержка

**Частые вопросы:**

**Q: Не приходит письмо с QR-кодом**
A: Проверьте папку "Спам". Если письма нет, QR-код можно найти на экране после оформления заказа (сделайте скриншот).

**Q: Забыл пароль от админки**
A: Обратитесь к разработчику для сброса пароля.

**Q: Не загружается изображение товара**
A: Проверьте размер файла (максимум 5MB) и формат (JPG, PNG, GIF, WebP).

**Q: Товар не отображается на сайте**
A: Проверьте, что товар сохранен и имеет остаток больше 0.

**Q: Как добавить 3D модель?**
A: Загрузите 3D модель в формате GLB на любой хостинг (например, Google Drive с публичным доступом) и вставьте ссылку в поле "URL 3D модели".
